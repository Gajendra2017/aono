What's IPFS?

"IPFS is a distributed file system that seeks to connect all computing devices with the same system of files. In some ways, this is similar to the original aims of the Web, but IPFS is actually more similar to a single bittorrent swarm exchanging git objects." - IPFS readme

Why IPFS?

Rather than requiring a centrally-controlled location for content to be hosted IPFS allows a distributed network to become content providers for each other, ideal for use cases such as static content delivery network.

What's with the huge hashes?

The hash is generated from the contents of the file and is guaranteed to always be the same. If the file changes then a new hash will be generated.

How can I use this?

As with other CDN's you can simply drop in a script tag on your page pointing at the resource you want to use.
